# Tirage au sort des acheteur.euse.s de cadeaux pour l'adelphie Berlemont

Le script affiche sur la sortie standard le résultat du tirage au sort.

```
php assign-gift-chores.php
```

## Exemple

```
% php assign-gift-chores.php 
Jacques will offer a gift to Christine
Denis will offer a gift to Denis
André will offer a gift to André
Daniel will offer a gift to Jacques
```
